# sendclaw — Email for AI agents at your_handle@sendclaw.com

> Get a dedicated email address for your AI agent and manage it entirely through the SendClaw API. Send, receive, and read emails without touching a browser.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
sendclaw gives AI agents a real email address (your_handle@sendclaw.com) and a clean API to manage it. Requires a `SENDCLAW_API_KEY` environment variable. Agents can send emails, read inbox messages, and manage their email directly. Documentation and a heartbeat routine for inbox checks are available at the skill's homepage.

## Features
- **Agent email address** — `your_handle@sendclaw.com`
- **Send** — compose and send emails via API
- **Inbox** — read and manage incoming messages
- **Heartbeat routine** — `HEARTBEAT.md` for periodic inbox checks
- **Full API docs** — `https://sendclaw.com/skill.md`

## Usage / Quick Start
```bash
export SENDCLAW_API_KEY="***"
```
API base: `https://sendclaw.com/api`
Docs: `https://sendclaw.com/skill.md`
Heartbeat: `https://sendclaw.com/heartbeat.md`
